🧠 Quiz App
A modern and interactive Quiz Application built using React that allows users to test their knowledge through multiple-choice questions and get instant feedback.

👉 Live demo : https://quizapp-zeta-one.vercel.app/

📌 Features
📝 Multiple-choice questions
✅ Instant answer validation
📊 Score tracking
🔄 Dynamic question rendering
⚡ Fast and responsive UI
💡 Clean and simple user experience

🛠️ Tech Stack
Frontend: React JS
Styling: CSS / Tailwind (update if needed)
Build Tool: Vite

🧠 What I Learned

While building this project, I revised and practiced:

React state management using useState
Handling events in React
Conditional rendering
Working with arrays and dynamic UI updates
Component-based architecture
